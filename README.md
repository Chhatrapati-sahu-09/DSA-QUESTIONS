# Data Structures and Algorithms (DSA) Questions & Solutions

This repository contains solutions to various Data Structures and Algorithms (DSA) problems implemented in Java. The solutions are structured by topic and difficulty level.

## Repository Structure

Currently, the repository is organized into the following categories:

* **[ARRAY-EASY](ARRAY-EASY/)**: A collection of 14 foundational array problems.
* **[ARRAY-MEDIUM](ARRAY-MEDIUM/)**: A collection of 14 medium-level array problems.

## 🛠️ How to Compile and Run

To compile and run any of the solutions, navigate to the respective directory and use the Java Development Kit (JDK) commands:

```bash
# Navigate to the topic directory
cd ARRAY-MEDIUM

# Compile the Java file
javac _01_Two_Sum.java

# Run the compiled class
java _01_Two_Sum
```
