# DSA Array Problems

This repository contains solutions to common Array problems implemented in Java. The solutions are structured in ascending order.

## Problems Covered

| # | Problem | Source Code | Description |
|---|---------|-------------|-------------|
| 01 | Largest Element | [_01_Largest_Element.java](_01_Largest_Element.java) | Finds the largest element in an array. |
| 02 | Second Largest Element | [_02_Second_Largest_Element.java](_02_Second_Largest_Element.java) | Finds the second largest element without sorting. |
| 03 | Check if Array is Sorted | [_03_Check_Array_Sorted.java](_03_Check_Array_Sorted.java) | Checks if the given array is sorted in non-decreasing order. |
| 04 | Remove Duplicates from Sorted Array | [_04_Remove_Duplicates.java](_04_Remove_Duplicates.java) | Removes duplicates in-place from a sorted array. |
| 05 | Left Rotate Array by One | [_05_Left_Rotate_By_One.java](_05_Left_Rotate_By_One.java) | Rotates the array elements to the left by one position. |
| 06 | Left Rotate Array by K Places | [_06_Left_Rotate_By_K.java](_06_Left_Rotate_By_K.java) | Rotates the array to the left by K positions. |
| 07 | Move Zeros to End | [_07_Move_Zeros_To_End.java](_07_Move_Zeros_To_End.java) | Moves all zeroes to the end of the array while maintaining relative order. |
| 08 | Linear Search | [_08_Linear_Search.java](_08_Linear_Search.java) | Searches for an element using the linear search algorithm. |
| 09 | Union of Two Sorted Arrays | [_09_Union_Two_Sorted_Arrays.java](_09_Union_Two_Sorted_Arrays.java) | Finds the union of two sorted arrays containing unique elements. |
| 10 | Find Missing Number | [_10_Find_Missing_Number.java](_10_Find_Missing_Number.java) | Identifies the single missing number in a range from 1 to N. |
| 11 | Maximum Consecutive Ones | [_11_Max_Consecutive_Ones.java](_11_Max_Consecutive_Ones.java) | Finds the maximum number of consecutive 1s in a binary array. |
| 12 | Single Number (Appears Once) | [_12_Single_Number.java](_12_Single_Number.java) | Finds the element that appears once in an array where every other element appears twice. |
| 13 | Longest Subarray with Sum K (Positives) | [_13_Longest_Subarray_Positive_K.java](_13_Longest_Subarray_Positive_K.java) | Finds the length of the longest subarray with a sum equal to K (positive numbers only). |
| 14 | Longest Subarray with Sum K (General) | [_14_Longest_Subarray_K.java](_14_Longest_Subarray_K.java) | Finds the length of the longest subarray with a sum equal to K (positives, negatives, and zeroes). |

## How to Run

To compile and run any of the solutions, use the Java Development Kit (JDK) commands in your terminal:

1. **Compile**:
   ```bash
   javac _01_Largest_Element.java
   ```

2. **Run**:
   ```bash
   java _01_Largest_Element
   ```
