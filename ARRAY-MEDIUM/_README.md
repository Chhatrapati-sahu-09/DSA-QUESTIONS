# DSA Array Problems (Medium)

This directory contains solutions to medium-level Array problems implemented in Java. The solutions are structured in ascending order and include both brute force and optimal approaches.

## Problems Covered

| # | Problem | Source Code | Description |
|---|---------|-------------|-------------|
| 01 | Two Sum | [_01_Two_Sum.java](_01_Two_Sum.java) | Finds two numbers in an array that add up to a specific target. |
| 02 | Sort Array of 0s, 1s, and 2s | [_02_Sort_Array_Of_0s_1s_And_2s.java](_02_Sort_Array_Of_0s_1s_And_2s.java) | Sorts an array containing 0s, 1s, and 2s in-place. |
| 03 | Majority Element (I) | [_03_Majority_Element_I.java](_03_Majority_Element_I.java) | Finds the element that appears more than N/2 times. |
| 04 | Kadane's Algorithm | [_04_Kadanes_Algorithm.java](_04_Kadanes_Algorithm.java) | Finds the contiguous subarray with the maximum sum. |
| 05 | Maximum Subarray Sum Extended | [_05_Maximum_Subarray_Sum_Extended.java](_05_Maximum_Subarray_Sum_Extended.java) | Prints the subarray that yields the maximum sum. |
| 06 | Stock Buy and Sell | [_06_Stock_Buy_And_Sell.java](_06_Stock_Buy_And_Sell.java) | Finds the maximum profit from buying and selling a stock. |
| 07 | Rearrange Array Elements by Sign | [_07_Rearrange_Array_Elements_By_Sign.java](_07_Rearrange_Array_Elements_By_Sign.java) | Rearranges elements into alternating positive and negative values. |
| 08 | Next Permutation | [_08_Next_Permutation.java](_08_Next_Permutation.java) | Finds the lexicographically next greater permutation of numbers. |
| 09 | Leaders in an Array | [_09_Leaders_In_An_Array.java](_09_Leaders_In_An_Array.java) | Finds all elements that are greater than all elements to their right. |
| 10 | Longest Consecutive Sequence | [_10_Longest_Consecutive_Sequence.java](_10_Longest_Consecutive_Sequence.java) | Finds the length of the longest consecutive elements sequence. |
| 11 | Set Matrix Zeroes | [_11_Set_Matrix_Zeroes.java](_11_Set_Matrix_Zeroes.java) | If an element in a matrix is 0, sets its entire row and column to 0. |
| 12 | Rotate Matrix by 90 Degrees | [_12_Rotate_Matrix_By_90_Degrees.java](_12_Rotate_Matrix_By_90_Degrees.java) | Rotates an N x N 2D matrix by 90 degrees clockwise in-place. |
| 13 | Print Matrix in Spiral Order | [_13_Print_Matrix_In_Spiral_Order.java](_13_Print_Matrix_In_Spiral_Order.java) | Returns all elements of the matrix in spiral order. |
| 14 | Count Subarrays with Given Sum | [_14_Count_Subarrays_With_Given_Sum.java](_14_Count_Subarrays_With_Given_Sum.java) | Finds the total number of continuous subarrays whose sum equals K. |

## How to Run

To compile and run any of the solutions, use the Java Development Kit (JDK) commands in your terminal:

1. **Compile**:
   ```bash
   javac _01_Two_Sum.java
   ```

2. **Run**:
   ```bash
   java _01_Two_Sum
   ```
